XYZ Tutions UI Design Project

Diploma in Sofware Engineering: UIF-UI Module Project | Designed Using Axure

Project Background

Suppose I have been approached by a Tuition Center XYZ Tutions. The tuition center has many teachers and many students. Teachers publish learning materials in a secured area. Each student belong to one class and each teacher belong to multiple class. The system should allow teachers to login and publish training materials. Students will be able to login and view the training materials.


Create a prototype for the home page, login page, login success page for both students and teachers. Teacher content posting page. Student content viewing page. Student Registration Page, Forget Password and Forget Password thank you pages.
